这篇文章用于简单归纳 Qwen-Image-Edit 2511 能完成的任务类型，方便快速查阅。

快速总览
#

分类 数量 说明
文字与排版类 15 文本增删改、纠错、版式贴合
局部外观编辑 25 局部 add/remove/modify，尽量不影响其他区域
全局语义编辑 10 允许整体像素变化但保持语义一致
人物一致性与多人物编辑 12 身份保持、多人物一致性
多图输入/多图融合 5 多图合成、合照与后续编辑
工业设计与几何推理 5 工业设计场景、结构线与工程表达
LoRA训练后扩展任务 23 借助 LoRA/微调增强稳定性与可控性
前沿探索与多模态扩展 6 3D意识、物理交互与跨表示编辑

合计：101 项任务。

标记规则
#

  • [S] 单图输入
  • [M] 多图输入（官方示例支持 image=[img1,img2]）
  • [ROI] 框选/局部区域编辑
  • [L] 调用（内置/外部）LoRA
  • [FT] 更偏向自训练 LoRA/微调

任务清单（101）
#

一、文字与排版类（15）
#

  1. [S][ROI] 在图中新增文字（招牌/海报/标签）
  2. [S][ROI] 删除既有文字（抹除并补全背景）
  3. [S][ROI] 替换文字内容（保持原位置语义一致）
  4. [S][ROI] 纠错：修正拼写/错字（可链式逐步纠错）
  5. [S][ROI] 局部改字：只改某几个字/某一处短语
  6. [S][ROI] 改单个字符属性（如只改某个字母颜色）
  7. [S][ROI] 保持字体/字号/风格前提下改文案
  8. [S][ROI] 中英双语文字编辑（同一图内混排也可）
  9. [S][ROI] 让新增文字与画面透视/光照一致（贴合场景）
  10. [S][ROI] 海报大标题改写且不破坏整体画面
  11. [S][ROI] 小字号细碎文本编辑（图内小区域文本）
  12. [S][ROI] 手写/书法类文本的局部修补（框选逐步修正）
  13. [S] “只改字不改图”式外观编辑（强调其他区域不动）
  14. [S][ROI] 对象上的铭牌/包装文字替换（尽量保持材质质感）
  15. [S][ROI] 文字元素的“加/删/改”组合编辑（多步骤串联）

（Qwen-Image-Edit 明确强调可做双语文字的加/删/改，并尽量保留原字体外观；也展示了框选逐步纠错的链式编辑流程。）

二、局部外观编辑（强调“其他区域尽量不变”）（25）
#

  1. [S][ROI] 添加一个新物体（如加指示牌/道具）
  2. [S][ROI] 添加物体并生成合理的反射/阴影/接触光
  3. [S][ROI] 删除一个显眼物体并补全背景
  4. [S][ROI] 删除细小杂物（如碎发/小遮挡物）
  5. [S][ROI] 替换物体（A→B），保持环境一致
  6. [S][ROI] 仅改变物体颜色（其余不动）
  7. [S][ROI] 仅改变物体材质（金属↔塑料↔木头等）
  8. [S][ROI] 仅改变物体纹理/图案（条纹/格纹/贴图）
  9. [S][ROI] 局部“修补/补洞/补缺口”（外观层面的修复）
  10. [S][ROI] 局部“去瑕疵/去污点/去反光斑”
  11. [S][ROI] 局部“更清晰/补细节”（轻度细化）
  12. [S][ROI] 只改某个区域的亮度/对比度/色相
  13. [S][ROI] 替换背景（人物/主体尽量保持）
  14. [S][ROI] 背景风格化而主体尽量不变（外观编辑取向）
  15. [S][ROI] 更换服装（外观编辑场景之一）
  16. [S][ROI] 给主体加配饰（眼镜/帽子/徽章等）
  17. [S][ROI] 去除配饰/局部装饰（不影响主体身份）
  18. [S][ROI] 让新增/替换元素与原图景深一致
  19. [S][ROI] 让新增/替换元素与原图噪点/锐度一致
  20. [S][ROI] 让新增元素遵循原图透视比例（不突兀）
  21. [S] 反复多轮编辑时尽量降低“越改越跑偏”（2511 强调减轻 drift）
  22. [S][ROI] 通过框选/区域提示实现“只改这一块”工作流
  23. [S][ROI] 在不重绘全图的情况下做“局部替换+局部修补”
  24. [S][ROI] 多个局部区域同时改（多处 add/remove/modify）
  25. [S][ROI] 用链式步骤把复杂编辑拆解（逐步逼近目标）

（以上“添加/删除/修改元素且其他区域尽量不变”的定义属于官方对“appearance editing”的描述，并有招牌插入、细小物体移除、改单个字母颜色、换背景/换衣等示例。）

三、全局语义编辑（允许整体像素变化但保持语义一致）（10）
#

  1. [S] 风格迁移（写实→动画/插画等）
  2. [S] 基于同一角色/主体做“IP 扩展/表情包化”
  3. [S] 保持主体身份与核心语义的“场景再创作”
  4. [S] “同一主体，多种主题变体”（语义一致，画面可整体变化）
  5. [S] 物体旋转（例如 90°/180° 视角翻转）
  6. [S] 新视角合成（novel view synthesis 作为语义编辑应用之一）
  7. [S] 让主体做“合理的动作/姿态变化”（语义层面变化，允许全局重绘）
  8. [S] 将同一人物转成“虚拟头像/二次元化”等语义风格重构
  9. [S] 将同一物体做“设计迭代版本”（语义保持，外观变化）
  10. [S] 面向“创意改写”的语义编辑（强调一致性而非逐像素保持）

四、人物一致性与多人物编辑（12）
#

  1. [S] 基于输入人像做“想象式编辑”但保持身份与视觉特征（2511 强调增强）
  2. [S] 同一人物在多次编辑中尽量保持一致（减少 identity drift）
  3. [S][ROI] 换发型/发色（保持身份）
  4. [S][ROI] 加/减妆容（保持身份）
  5. [S][ROI] 表情微调（保持身份）
  6. [S][ROI] 加/换衣服（保持身份；官方点名应用场景之一）
  7. [S][ROI] 换背景（保持人物主体；官方点名应用场景之一）
  8. [S][ROI] 加/换配饰（保持身份）
  9. [S][ROI] 去除遮挡/小杂物（如碎发）并保持脸部/主体一致
  10. [S] 多人物场景里对某一个人局部编辑、尽量不影响其他人
  11. [S] 多人物整体一致性更好（2511 强调 multi-person consistency）
  12. [S] 把多人物照片做“统一风格/统一光照”并保持身份可辨

五、多图输入/多图融合（5）
#

  1. [M] 输入两张图并生成“同框合成/合照构图”（官方示例 image=[img1,img2]）
  2. [M] 两个主体在同一画面中产生交互式构图（示例 prompt 描述“左/右…相对”）
  3. [M] “两张不同人物图 → 高保真合照融合”（2511 明确强调）
  4. [M] 多图融合后继续做局部编辑（合照里加/删/改元素）
  5. [M] 多图融合后做风格/光照一致化（减少拼贴感）

六、工业设计与几何推理（5）
#

  1. [S] 批量工业产品概念图/变体生成（2511 强调更强）
  2. [S] 工业部件/产品的材质替换（工程场景）
  3. [S] 工业设计场景中“保持结构语义但换外观方案”（系列化出图）
  4. [S] 直接生成用于设计/标注的辅助构造线（geometric reasoning）
  5. [S] 生成“设计草图/结构线 + 渲染图”的混合表达（偏工程表达）

（上面 1–72 属于“官方明确描述/示例覆盖的能力范畴”）

七、LoRA训练后更容易稳定的扩展任务（23）
#

  1. [L] 真实感光照增强/光照控制
  2. [L] “生成新视角/多角度”能力增强
  3. [L] 超明确的多机位/多角度相机控制
  4. [L] 多角度结果更接近 3D 一致（多角度 LoRA 训练于 3D 渲染数据）
  5. [L] 姿态迁移/姿势跟随（社区常称 AnyPose / pose transfer）
  6. [L] 写真→动画/二次元转换（photo-to-anime LoRA 方向）
  7. [L] 漫画网点/黑白漫画化（manga tone LoRA 方向）
  8. [L] 特定风格迁移 LoRA（style transfer LoRA 方向）
  9. [L] “光照迁移/光源变化”更强（light migration LoRA 方向）
  10. [L] 3D 方向性光源控制（社区 demo 指向 3D lighting control）
  11. [L] 近似“无损放大到 ~2K”的放大 LoRA（明确以 2511 训练、用于放大）
  12. [L] 更高分辨率/更大画幅的稳定输出（常与放大/细节 LoRA 结合）
  13. [L] 更快推理（如 4-step Lightning 加速 LoRA，在 ComfyUI 教程里作为可选项出现）
  14. [FT] 训练你自己的“品牌视觉/统一画风”LoRA（企业视觉规范、固定配色/材质语言）
  15. [FT] 训练你自己的“角色 IP 一致性”LoRA（固定人设跨场景产出）
  16. [FT] 训练你自己的“服装/穿搭风格库”LoRA（某品牌/某版型的稳定换装）
  17. [FT] 训练你自己的“产品系列化外观”LoRA（同一产品多材质/多配色/多细节件）
  18. [FT] 训练“工业部件材质库”LoRA（螺栓/外壳/电镀/喷砂等材质可控）
  19. [FT] 训练“字体/版式模板”LoRA（特定招牌/海报模板的稳定文本编辑外观）
  20. [FT] 训练“特定相机系统/镜头语言”LoRA（仰拍/俯拍/焦段风格一致）
  21. [FT] 训练“特定光照棚拍方案”LoRA（产品棚拍多灯位风格一致）
  22. [FT] 训练“特定数据域修复”LoRA（例如某类瑕疵/噪声/压缩痕的定向修复）
  23. [FT] 训练“特定结构线/标注风格”LoRA（工程制图风格的辅助线/标注更统一）

八、前沿探索与多模态扩展（3D意识与物理交互）（6）
#

（这部分代表了突破纯 2D 像素映射，向具备 3D 意识、物理常识和多模态转换演进的最新 AI 路线。这类任务通常超越了单纯的“外观”与“语义”编辑。）

  1. [S][ROI] 基于拖拽的像素级精准空间形变控制（Drag-based manipulation，例如精准拉长轴距或改变角色关节角度，且保持物理结构合理）
  2. [S] 突破画幅边缘的无缝图像外扩与全景生成（Outpainting / Canvas Extrapolation，保持原有透视与光照向外延伸）
  3. [S] 基于单图提取 3D 深度/法线并进行“物理重打光”（Physically-based Relighting，在场景中添加/移动虚拟 3D 光源并产生正确的物理阴影）
  4. [S] 赋予静态图“运动频谱”，生成支持鼠标交互的动态场（Generative Image Dynamics，使画面元素具备真实的物理晃动与回弹属性）
  5. [S] 在编辑全链路中注入/提取不可见的溯源水印（Steganography / 无损注入不可见版权或 AI 防伪标识）
  6. [S] 将生成的像素光栅图解构并转换为可无限放大的矢量图层（Raster-to-SVG / 提取数学曲线与节点，实现工程化节点级编辑）
